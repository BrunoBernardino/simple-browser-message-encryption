# Simple Browser Message Encryption

> [!IMPORTANT]
> This project is outdated and no longer maintained or deployed.

I built this tool so I could easily send an encrypted message/text via unsafe channels (and decrypt too) in a fairly secure way.

I always prefer PGP, but that requires technical knowledge from the other person I'm communicating with, and this doesn't.

This repo is originally kept on [GitHub](https://github.com/BrunoBernardino/simple-browser-message-encryption), and mirrored to [Codeberg](https://codeberg.org/BrunoBernardino/simple-browser-message-encryption).

## Development

View locally using `deno run --allow-net --allow-read jsr:@std/http@1.1.0/file-server src` or `npx serve@14.2.6 src`.

## Deployment

Just push to the `main` branch.
